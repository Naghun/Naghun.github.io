Duvar Apartments

Technologies

Wordpress, HTML&CSS(SASS), Javascript

About Project

Insternship project done with wordpress understrap theme developed via coding

Fully functional Apartment Rental page developed using WordPress and PHP. The design is crafted with CSS (SCSS), while interactivity is handled through jQuery and JavaScript. The API interactions are powered by Ajax.

The project features pages about the location of the apartments and a dedicated page for blogs about attractions in Blagaj. The contact form and reservation details are stored in the database as separate post types.

Once you submit your reservation dates and contact information, the reservations are sent as drafts awaiting an administrator's approval. Once administrator publish them, the reservation is completed, and a confirmation email is sent to your inbox.

Tournament Game

Technologies

Django, HTML & CSS, Photoshop

About Project

First significant solo project, developed exclusively in Django. I had plans to transition it to React Django in the future and give it a redesign. A tournament card game focused on intricate battle mechanics, featuring a diverse array of fighters from four different races.

The landing page requires you to log in before proceeding to the game. Once inside, the game page boasts an animated background and options to start a new game or load previously saved worlds. Each world begins on day one, with each tournament adding another day to its timeline.

Champions are categorized into four tiers and four races, where champions of all tiers, regardless of strength, have the potential to excel, although some may require more luck than others. Tournaments consist of three phases: the Draft Phase, Bonus Phase (where champions can receive buffs or nerfs due to RNG), and the Tournament Phase. Fatigue carries over from one fight to the next, making tired fighters slower and more vulnerable, thus giving fresher warriors a chance to prevail.

With over 1000 lines of code dedicated solely to game mechanics, this project is one that I was committed to and in future I aim to develop into something worth boasting about. I hope to address the lack of designs and give this game from my childhood the vibrant colors it deserves.
